## 😺 프로젝트 소개
* 고롱고롱 : 블록체인 기반 길고양이 개인 기부 플랫폼
* 고양이의 **고롱고롱 (golong)** 기분 좋은 소리가 **오래갈 수 있도록 (golong)**
* 길고양이 개인 구조에 대한 기부과정을 고롱고롱 플랫폼에서 투명성있고 편리하게 진행할 수 있습니다.
